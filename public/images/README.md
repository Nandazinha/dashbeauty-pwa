# Imagens do DashBeauty PWA

## Ícones necessários

Baixe os ícones em diferentes tamanhos e coloque nesta pasta:

- icon-72.png (72x72 pixels)
- icon-96.png (96x96 pixels)
- icon-128.png (128x128 pixels)
- icon-144.png (144x144 pixels)
- icon-152.png (152x152 pixels)
- icon-192.png (192x192 pixels)
- icon-384.png (384x384 pixels)
- icon-512.png (512x512 pixels)

## Gerador de ícones

Use https://favicon.io/ ou https://www.pwabuilder.com/ para gerar os ícones.

## Ícones alternativos

Você pode usar emojis ou Font Awesome enquanto não tem as imagens reais.
